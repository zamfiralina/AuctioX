Proiectul AuctioX consta intr-o aplicatie Web ce pune la dispozitie utilizatorilor un sistem de licitatii a unor produse cu caracter personal, pentru care se poate licita pana cand vanzatorul decide sa Ã®nchida licitatia respectiva, obiectul revenindu-i ofertantului celei mai mari sume.

Acest document vizeaza detalierea modului in care aplicaia noastra functioneaza. Fiecare utilizator poate propune o licitatie sau poate licita la una deja existenta. Aplicatia prezinta un mod foarte simplu de folosire pentru utilizator (este un site asemantor E-bay). Utilizatorul isi va crea un cont, fiind nevoit sa completeze date personale precum nume, prenume, data nasterii, tara si orasul in care locuieste. Dupa ce si-a creat contul utilizatorul ppoate introduce o noua licitatie oferind diverse informatii despre aceasta (numele obiectului licitat, pretul de start, data expirarii licitatiei). Ulterior un alt utilizator inregistrat in aplicatie poate licita pentru obiectul respectiv.

In implementarea functionalitatilor din partea de backend am folosit limbajul de programare Python. Transmiterea de date intre partea de frontend (html si css) si partea de backend se face prin functii javascript si ajax care preiau datele introduse de utilizator si fac un reqest(cerere) catre server cu datele respective. Serverul va sti sa interpreteze in mod corespunzator fiecare cerere, va executa anumite operatii si ii va transmite inapoi utilizatorului(daca este cazul) fie un mesaj de succes fie un mesaj de eroare.
